# 📊 Netflix Dashboard Analysis with Python & Seaborn

## 📌 Project Overview

This project performs data cleaning, preprocessing, analysis, and visualization on the Netflix Titles dataset using **Python**, **Pandas**, **Matplotlib**, and **Seaborn**. The final output is an interactive dashboard containing multiple visualizations that provide insights into Netflix content.

---

## 🚀 Features

* Load and inspect the Netflix dataset
* Handle missing values
* Remove duplicate records
* Correct data types
* Export cleaned dataset
* Create a professional dashboard with multiple charts

---

## 📊 Dashboard Visualizations

* 🎬 Movies vs TV Shows Distribution
* ⭐ Content Ratings Distribution
* 🌍 Top 10 Countries by Number of Titles
* 🎭 Top 10 Genres
* 📈 Content Released Per Year
* 📅 Content Added to Netflix Per Year

## 📈 Key Insights

* Movies make up the majority of Netflix content.
* TV-MA is the most common content rating.
* The United States has the highest number of Netflix titles.
* Drama is the most popular genre.
* Netflix content production increased rapidly after 2015.
* The number of titles added to Netflix peaked around 2019.

## ▶️ How to Run

1. Clone this repository.
2. Install the required libraries:

```
pip install pandas matplotlib seaborn
```

3. Run the Python script:

```
python netflix_dashboard.py
```
